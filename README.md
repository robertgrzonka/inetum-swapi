## Installation
1. Clone the repo:
   ```sh
   git clone https://github.com/robertgrzonka/inetum-swapi.git
   ```
2. Install dependencies:
   ```sh
    npm install
    ```
3. Start the app:
    ```sh
    npm start
    ```
4. Open [http://localhost:5173/](http://localhost:5173/) to view it in the browser.
